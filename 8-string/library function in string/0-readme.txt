String Library Function in C:
    1) Printf() & scanf()		in scanf() string break at white spaces
    2) puts() & gets ()			in gets() string continue at white spaces
    3) strlen() & strnlen() 		length of string
    4) strcmp() & strncmp()		compare the string
    5) strcat() & strncat()		concatenate mean combine the string
    6) strcpy() & strncpy()		copy the string
    7) strchr() & strrchr()		start the string at specific character 
    8) strstr() 			start the string from other string
    9) strlwr() 			convert the string into lower case
    10) strupr() 			convert the string into upper case
    11) strrev() 			reverse the given string
    12) strset() & strnstr() 		convert the string into a specific character
    13) strdup()			duplicate the string	
    14) strspy() & strcspy()            for length of string on the base of character on other string
    15) strpbrk() 		        print first character of 2nd string present in 1st 
    16) strxfrm()		        transforms the first n characters of the string src into current locale and place them in the string dest.	
    17) strtok()		        It returns a pointer to the first character of a token or to a null pointer if there is no token.	
    18) strerror()			it gives error if file present.
    19) memcpy()			copy the string
    20) memchr()			start the string at specific character
    21) memcmp()
    22) memmove()			move the string into other string
    23) memset()			convert the string into a specific character
    24) strcoll()
